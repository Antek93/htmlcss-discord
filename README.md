In questo progetto ho approfondito diversi concetti fondamentali di HTML e CSS, applicandoli per ricreare l'interfaccia grafica di Discord. In particolare, ho imparato a:

Strutturare correttamente una pagina web utilizzando un layout complesso e suddiviso in sezioni.
Utilizzare il box model per organizzare spazi, margini, padding e dimensioni degli elementi.
Gestire layout flessibili tramite Flexbox, creando sidebar, contenuti principali e colonne responsive.
Applicare una corretta gerarchia tipografica con testi, icone e liste.
Lavorare con colori, sfondi e temi scuri, mantenendo coerenza con il design originale.
Migliorare la semantica del codice attraverso tag HTML appropriati.
Organizzare il progetto con una struttura ordinata di cartelle e file CSS.


Tecnologie utilizzate:

HTML5 – per creare la struttura dell’interfaccia utente.
CSS3 – per lo styling e la riproduzione dell'aspetto grafico di Discord.
Flexbox – per la gestione del layout a più colonne e delle sidebar.
Google Fonts / Icon set – per arricchire la tipografia e le icone del progetto.
Git & GitHub – per versioning e pubblicazione del repository.

In this project, I explored several fundamental HTML and CSS concepts while recreating the visual interface of Discord. In particular, I learned how to:

Properly structure a web page using a complex, multi‑section layout.
Use the box model to manage spacing, margins, padding, and element dimensions.
Build flexible layouts with Flexbox, including sidebars, main content areas, and responsive columns.
Apply a clear typographic hierarchy with text elements, icons, and lists.
Work with colors, backgrounds, and dark themes, ensuring consistency with the original Discord design.
Improve code semantics through appropriate HTML tags.
Organize a project using a clean folder structure and well‑maintained CSS files.


Technologies Used:

HTML5 – to create the structure of the user interface.
CSS3 – to style and visually reproduce Discord’s UI.
Flexbox – to handle multi‑column layout and sidebar positioning.
Google Fonts / Icon Sets – to enhance typography and iconography.
Git & GitHub – for version control and repository publishing.
